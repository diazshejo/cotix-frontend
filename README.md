\# Cotix 🚀



Plataforma SaaS para gestión de cotizaciones, clientes y productos.



\## 🧱 Arquitectura



\- Backend: Django + DRF

\- Frontend: Next.js

\- Base de datos: PostgreSQL (planeado)

\- Autenticación: JWT



\## 📦 Módulos



\- Clientes

\- Productos / Servicios

\- Cotizaciones

\- Usuarios / Roles (multi-tenant)
